Save as PDF Options

This extension uses the Pdfcrowd online service to create PDFs from web pages. If you want to change the default settings:

  1. Sign up for a Premium License at Pdfcrowd
  2. Customize the settings on the Pdfcrowd web site
  3. Click the Save Settings button

You can find more information about this extension at http://pdfcrowd.com/save-as-pdf-explained/.
